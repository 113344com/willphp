# WillPHPv4

WillPHPv4是一个轻量级组件化php开发框架

####环境要求

- PHP版本需要 >=5.6
- PDO扩展

####安装使用

使用 composer 命令进行安装：

    composer create-project willphp/willphp blog --prefer-dist


####开发手册

开发手册：[https://willphp.gitee.io/](https://willphp.gitee.io/)

####新版特性

1. 组件化：可使用composer组件。
2. 轻量级：核心系统只需要一些非常小的库。
3. ORM库：实现对象关系数据库映射。
4. 可扩展：可自定义框架组件、类库、函数。
