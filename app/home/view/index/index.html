开始作用

  1. 安装环境：安装PHP环境，用cmd进入 www目录
  2. 安装项目：composer create-project willphp/willphp blog --prefer-dist (blog是自定义的项目名)
  3. 进入项目：cd blog 重命名.env.example：rename .env.example .env
  4. 修改配置：打开.env文件修改数据库配置
  5. 导入数据：使用phpmyadmin等工具导入测试表：data/blog.sql
  6. 访问地址：http://localhost/blog/public
